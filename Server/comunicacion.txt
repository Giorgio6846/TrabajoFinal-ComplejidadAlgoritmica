La idea de la comunicacion entre la app y el servidor
Se realiza mediante comunicacion como

Caso 1:
Envie los departamentos
Mensaje App: Departamentos
Mensaje Servidor: JSON departamentos

Caso 2:
Envie todas las provincias
Mensaje App: TProv
Mensaje Servidor: JSON todas las provincias

Caso 3:
Envie las provincias de un departamentos
Mensaje App: Provincias.Departamento
Mensaje Servidor: JSON provincias departamentos

Caso 4:
Envie todos los distritos
Mensaje App: TDistr
Mensaje Servidor: JSON todos los distritos

Caso 3:
Envie los distritos de una provncia
Mensaje App: Distrito.Provincias.Departamento
Mensaje Servidor: JSON distrito

etc
